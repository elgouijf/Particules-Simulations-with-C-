Detailed Description

Représente une cellule de la grille spatiale utilisée pour accélérer le calcul des interactions.

Une cellule contient un ensemble de particules ainsi qu'une liste de cellules voisines. Cette structure permet de limiter le calcul des interactions aux particules situées dans la cellule courante et dans les cellules adjacentes.
