Detailed Description

Représente l'ensemble du système physique simulé.

L'univers contient toutes les particules ainsi qu'une grille de cellules permettant d'accélérer le calcul des interactions de Lennard-Jones.

Il stocke également les paramètres physiques et géométriques du problème :

  • distance de coupure,
  • dimension,
  • tailles du domaine,
  • nombre de cellules par direction,
  • paramètres du potentiel de Lennard-Jones.

◆ calcule_forces()

void univers::calcule_forces ( )

Calcule les forces exercées sur toutes les particules.

Calcule les forces de Lennard-Jones sur toutes les particules.

Les interactions sont limitées aux particules situées dans la cellule courante et dans ses voisines, avec une coupure à la distance r_cut.

Cette implémentation exploite la décomposition spatiale en cellules afin de limiter le calcul aux paires de particules potentiellement proches.

Optimisations utilisées :

  • calcul uniquement avec des distances au carré,
  • absence de sqrt et de pow,
  • calcul direct sur les composantes dx, dy, dz,
  • application du principe action-réaction.

Les interactions sont tronquées à la distance de coupure r_cut.

La force utilisée correspond à une interaction de type Lennard-Jones.
